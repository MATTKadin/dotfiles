# Dotfiles

Repository enthält Dotfiles.  
werden mit [GNU Stow](https://www.gnu.org/software/stow/) verwaltet und über GitHub synchronisiert.

---

## Voraussetzungen

installiere benötigte Programme:

Debian/Ubuntu
```bash
sudo apt install git stow 

```

ArchLinux
```bash
sudo pacman -S git stow
```

Fedora/Redhat
```bash
sudo dnf install git stow
```

NerdFonts:
```bash
https://github.com/ryanoasis/nerd-fonts/releases/download/v3.4.0/GeistMono.zip
```
